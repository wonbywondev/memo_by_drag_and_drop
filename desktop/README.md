## macOS 데스크톱 앱 실행 방법

이 폴더에는 `PRD_for_desktop.md`를 바탕으로 한 **로컬 Markdown 기반 GTD·PARA 데스크톱 앱(프로토타입)** 이 들어 있습니다.

### 1. 의존성 설치

```bash
cd /Users/won/dev/02_WIP/drag_drop_codex/desktop
npm install
```

> `package.json` 에는 Electron 이 devDependency 로 정의되어 있습니다.

### 2. 앱 실행

```bash
npm start
```

Electron 이 실행되며, 메인 윈도우가 열립니다.

### 3. 현재 구현된 기능

- **루트 폴더 선택**
  - 상단의 `루트 폴더 선택` 버튼 → macOS 폴더 선택 다이얼로그 → 선택한 폴더가 루트로 설정
- **Markdown 파일 인덱싱**
  - 루트 폴더 이하의 모든 `.md` 파일을 재귀적으로 스캔해서 좌측 리스트에 표시
- **노트 열기/미리보기**
  - 좌측에서 파일 클릭 시 우측 에디터 + 미리보기에서 내용 확인
- **노트 수정 및 저장**
  - 내용을 수정 후 `저장 (Cmd+S)` 버튼 또는 `Cmd+S` 단축키로 원본 `.md` 파일에 직접 저장
- **새 노트 생성**
  - `새 노트(.md)` 버튼 또는 `Cmd+N` → 파일명 입력 → 루트 폴더 바로 아래에 Markdown 템플릿 파일 생성

### 4. 파일 구조

- `main.js`: Electron 메인 프로세스. 윈도우 생성, 폴더 선택/파일 읽기·쓰기 IPC 처리.
- `preload.js`: Renderer 에서 사용할 `desktopApi` 브리지 정의.
- `index.html`: 메인 UI (좌측 목록 + 우측 편집/미리보기).
- `renderer.js`: 클라이언트 로직 (폴더 선택, 파일 목록/열기/저장, 미리보기 렌더링).

### 5. 앞으로 확장 포인트

- A/B 클래스 구분 표시, 프로젝트/목표/영역 트리 뷰
- 관계(연결) 편집 UI 및 프런트매터 업데이트
- 키보드 단축키 추가(Cmd+K 검색 등), 타임라인 뷰
- 멀티 윈도우(검색 전용/타임라인 전용) 지원
